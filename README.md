# TotalEnvios-App


### Prerequisites

Npm i ionic -g
npm i cordova -g


### Installing

npm install
ionic cordova platform add android@7.1.4
ionic cordova build android



* [Ionic](https://ionicframework.com/docs/v3) - The web framework used
